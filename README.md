# ChordLine

An app to revolutionize the experience of local live musicians.

## Local Dev

cd apps/web
npm install (one time)
npm run dev

## Tech Stack

Web Hosting: Vercel
Backend Hosting: Render
Database: Supabase
Auth: Clerk

Web Framework: NextJS
Web UI: ShadCN + Tailwind

Testing: Playwright
Package Manager: pnpm
